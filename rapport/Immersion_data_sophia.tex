\chapter{Méthode}\label{mtd}

Présenter et justifier qualitativement et (ou) quantitativement :

\begin{itemize}
\vspace{9pt}
\item les hypothèses formulées (s'il y a lieu) ;
\vspace{9pt}
\item les variables écartées (sur quelle base?) ;
\vspace{9pt}
\item les cas exclus (justifier) ;
\vspace{9pt}
\item les variables construites (s'il y en a) ; 
\vspace{9pt}
\item les tests statistiques conduits (s'il y a lieu) ;
\vspace{9pt}
\item le type de modèle utilisé.
\end{itemize}

\vspace{9pt}
Vous êtes invités à décrire formellement vos résultats à l'aide de l'environnement équation : \\
\vspace{9pt}
\begin{equation*}
\begin{aligned}
\text{Modèle qui tue}\, (x_{i_{0}}) \quad & = \underset{\text{modèle }\in \mathcal{M}}{\max}\,\text{modèle}(x_{i_{0}}) \\
& = \frac{\mathrm{e}^{f(x_{i_{0}})}}{\sum_{i=1}^{d} \mathrm{e}^{f(x_{i})}} + \quad \text{plein d'autres choses}
\end{aligned}
\end{equation*}
